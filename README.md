Digital Skills Academy Website
==============================

Course: Web Development Fundamentals
Assignment: Build a Multi-Page Educational Website
Submission Deadline: Tuesday 9th June, 3:00 p.m

-----------------------------------
Project Overview
-----------------------------------
This project is a responsive educational website created for the Digital Skills Training Academy. 
It demonstrates the use of HTML structure, hyperlinks, images, and CSS styling techniques.

The website includes:
- Home Page (index.html)
- About Page (about.html)
- Courses Page (courses.html)
- Contact Page (contact.html)

-----------------------------------
Features
-----------------------------------
- Semantic HTML tags: <header>, <nav>, <main>, <section>, <article>, <footer>
- Internal navigation links between all pages
- External links to at least 3 websites (Facebook, Twitter, LinkedIn)
- Minimum of 5 images with alt attributes
- Responsive design using Flexbox and CSS Grid
- Custom styling: colors, backgrounds, borders, shadows, rounded corners
